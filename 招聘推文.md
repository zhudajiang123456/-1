# 🤖 全国机器人实验室智能导航系统 · 核心团队招募

---

## 我们在做什么

我们正在构建中国首个**机器人领域科研资源智能导航平台**——覆盖全国 320+ 顶级机器人实验室，整合导师信息、研究方向、成果数据，用 AI 帮助每一位有志于机器人研究的学生，找到最适合自己的科研归宿。

这不是一个普通的信息聚合网站。我们相信，**一个好的导向，能改变一个人的研究轨迹，乃至整个领域的走向。**

---

## 我们在找谁

### 前端工程师（1 名）

**你将做什么**
- 打造高性能、强交互的科研导航体验（地图可视化、智能搜索、动态筛选）
- 与设计师紧密协作，将像素级设计还原为流畅的用户界面

**我们希望你**
- 熟练使用 HTML / CSS / JavaScript，有 Vue 或 React 项目经验
- 对数据可视化（ECharts、D3.js 等）有实践经验者优先
- 热爱机器人、AI 或科研方向加分

---

### 全栈 / 后端工程师（1 名）

**你将做什么**
- 构建实验室数据库 API 与搜索引擎
- 设计用户行为分析与个性化推荐系统

**我们希望你**
- 有 Python / Node.js 后端开发经验
- 了解 Elasticsearch 或向量检索等搜索技术者优先
- 有过开源项目贡献或独立项目经历者优先

---

### 内容运营 / 数据研究员（1 名）

**你将做什么**
- 持续更新、核实全国机器人实验室核心数据（导师、成果、招生信息）
- 撰写科研选择指南、实验室深度报道等内容

**我们希望你**
- 对机器人、人工智能、自动化领域有浓厚兴趣
- 具备基本的数据整理能力（Excel / Python / 爬虫皆可）
- 文字表达清晰，能独立撰写高质量中文内容
